# Deployment: Part 3

This project is provided as a starting point for deploying an Express app that uses a MySQL database with Heroku.

## Running the app

- Create a local database.
- Create a file `.env` with the environment variables from `.env.example`. `DB_DATABASE` will be the name of the database you created.
- Run `npm install` to install dependencies.
- Run `npm run import` to intialize the database.
- Start the project by running `npm start`.
